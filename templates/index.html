About UnShiney

UnShiney is a deep learning project that uses neural networks to remove shine, glare, and reflections from images. The model takes grayscale images as input and produces processed images with reduced shine.

This web demo showcases several model architectures:

  • Fully Connected: A dense neural network that processes the entire image at once
  • Convolutional: Uses convolutional layers to process local patterns in the image
  • Hybrid: Combines convolutional and dense layers for processing
